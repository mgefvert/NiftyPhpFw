13. Other useful functions and classes

These are some common utility functions and classes that helps when doing stuff in PHP.

13.1 (Mostly) global functions

Function Description
a($obj, $key, $default=null) Handy function for the very common case of
isset($obj->$key) ? $obj->$key : $default

Works on both arrays and objects.
html($text) A wrapper around htmlspecialchars($text, ENT_QUOTES). Escapes all text output for safety against CSS attacks. A real work-horse.
pr($var) If you ever want to dump values onscreen (for debugging), use this function.
<?= pr($variable) ?> or die(pr($variable)); writes a clear and concise output of the particular object, value or array that you need to inspect.
_t($text) Translates a string using NF_Translate_CSV. If no text matches the input, just pass $text through.
$Path Provides properties for the different paths in the Nifty applications, such as /app, /public, and so on.
NF_DB::hash(array $array, $field) Hashes an array of arrays or array of objects according to the field $field. Useful when you have a group of objects in sequential key order, and you need the keys to be the values of a specific field instead (like, user ID or something).
NF_DB::group(array $array, $field) Groups an array of arrays or array of objects according to the field $field. Makes groups quickly and harmlessly - extremely useful when grouping objects for output according to customer name, month and year, or something similar.

13.2 Other useful classes

Class Description
NF_IMAP Class that allows you to work easily with IMAP connections and mailboxes.
NF_Net Makes HTTP requests easy.
NF_RSS, NF_RSS_Item Allows you to easily create or consume RSS feeds.
NF_SendMail Since PHP's own sendmail() function is rather difficult to get precisely right, NF_SendMail encapsulates that and provides an ability to add cc:, bcc:, HTML text and attachments easily.
NF_Filter An easy way of filtering input. I especially like NF_Filter::fint() that can force several variables at once to integers.
NF_Text A few nice functions to convert text, mostly between ANSI and Unicode, using the best services available on the system.
NF_GD Encapsulates the GD graphics library and helps with drawing and image manipulation.
NF_Password Provides encrypting, decryption and password handling. Since it's difficult to get these things absolutely right (and the grave implications for when you get it wrong), Nifty does it for you. Can encrypt passwords and compare passwords, using the best methods available on the system.
NF_Upload Makes it easy to handle file uploads. Just wrap NF_Upload around the name of the upload form field, and there you go.
